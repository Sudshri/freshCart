# 🛒 FreshCart Grocery App

FreshCart is a premium grocery shopping backend application built using Node.js and Express. It provides a simple, fast, and scalable REST API for managing grocery items.

---

## 🚀 Features
- Full CRUD operations for grocery items
- RESTful API design
- Lightweight and fast Express server
- Unique ID generation using UUID
- CORS enabled for frontend integration

---

## 📦 Tech Stack
- Node.js
- Express.js
- UUID
- CORS

## ⚙️ Installation & Setup

1. Clone Repository
git clone https://github.com/your-username/freshcart-grocery.git

2. Go to Project Folder
cd freshcart-grocery

3. Install Dependencies
npm install

---

## ▶️ Run Application

Start Server:
npm start

Development Mode:
npm run dev

Server URL:
http://localhost:3000

---

## 📡 API Endpoints

GET /items       → Get all items  
POST /items      → Add new item  
PUT /items/:id   → Update item  
DELETE /items/:id → Delete item  

---

## 🧪 Sample Request (POST /items)

{
  "name": "Apple",
  "price": 120,
  "quantity": 10
}
